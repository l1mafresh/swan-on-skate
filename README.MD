# Лебідь на скейті (Лебідь скейтбордист)
Гра про Лебедя, де треба їздити на скейті та збирати монети. За монети можна купити скіни та локації. Орієнтовано на мобільні пристрої, але можна запустити і на ПК.

## Технології
Ця гра зроблена в Snail IDE (мод Scratch, TurboWarp та PenguinMod; snail-ide.com).

## Завантаження
Завантажте файл HTML, який працює на будь-яких пристроях і без інтернету. Якщо Ви хочете редагувати гру в Snail IDE, завантажте файл .snail. Якщо запитає, за допомогою якої програми відкрити HTML-файл, виберіть Chrome, Opera чи інший браузер.

## Як грати?
Треба водити Лебедя пальцем (чи мишкою), оминати перешкоди і збирати монети. За монети можна купити скіни та локації.
